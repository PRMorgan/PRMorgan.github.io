AI-Kido
AI-Kido is a 2D figthing game in which we pit two computer players against one another. Red Fighter is implemented using random event generation while Blue Fighter is implemented using a neural network and genetic algorithm.
5e Character Creator
5e Character Creator is a command line tool that walks you through the steps of character creation for Fifth Edition Dungeons and Dragons. Once you have built your SRD-compliant character, the tool will create a few conveniently organized files with everything you need to play the game minus the dice, of course!
